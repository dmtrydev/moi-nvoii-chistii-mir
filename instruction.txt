ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ СКРИПТА

1. УСТАНОВКА ЗАВИСИМОСТЕЙ

Установи необходимые библиотеки:

pip install curl_cffi

Если нет pip:

sudo pacman -S python-pip

2. ПОДГОТОВКА ФАЙЛОВ

В одной папке должны находиться:

* main.py (скрипт)
* INNS.txt (список ИНН)

Формат INNS.txt:
Каждый ИНН с новой строки, только цифры:

7707083893
7710140679
7801234567

Строки вида 0000000000 автоматически игнорируются.

3. ЗАПУСК СКРИПТА

В терминале перейди в папку со скриптом и выполни:

python main.py

4. ЧТО ДЕЛАЕТ СКРИПТ

* Читает ИНН из INNS.txt
* Отправляет запросы на сайт
* Работает в несколько потоков (по умолчанию 10)
* Сохраняет результаты

5. РЕЗУЛЬТАТ

После выполнения появятся файлы:

licenses.json — содержит найденные данные
run.log — лог выполнения (ошибки, прогресс)

6. НАСТРОЙКА СКОРОСТИ

В файле main.py есть параметр:

THREADS = 10

Можно изменить:

* 5 — если слабый интернет
* 20–50 — быстрее, но возможны блокировки

7. ВОЗМОЖНЫЕ ПРОБЛЕМЫ

* Если много "брак" — уменьшить THREADS
* Если ошибки сети — проверить интернет
* Если скрипт падает — смотреть run.log

8. КРАТКО

9. Подготовить INNS.txt

10. Установить зависимости

11. Запустить: python main.py

12. Получить результат в licenses.json

Готово.
